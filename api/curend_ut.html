template<int N, int M, typename Aux, typename Func>
inline auto curend::ut::transform(Func const &f, glm::vec<N, float> const &mu, glm::mat<N, N, float> const &sqrt_covar, const float &alpha = 0.1f, const float &beta = 2.0f, const float &kappa = 0.0f) -> UnscentedTransformResult<M, Aux>

Performs the Unscented Transform (UT) on a nonlinear function.

The Unscented Transform is a method for calculating the statistics of a random variable which undergoes a nonlinear transformation. It uses a set of carefully chosen sample points (sigma points) to capture the mean and covariance of the transformed distribution.

Note

The function f must return a tuple of (transformed_point, valid_flag) where valid_flag indicates if the transformation was successful. If any sigma point transformation fails, the entire transform will return false.

Template Parameters:

  • N – Input dimension of the function

  • M – Output dimension of the function

  • Func – Function type that takes a glm::vec<N, float> and returns a tuple of (glm::vec<M, float>, bool)

Parameters:

  • f – The nonlinear function to transform. Must return a tuple of (transformed_point, valid_flag, aux_data)

  • mu – Mean of the input distribution

  • sqrt_covar – Square root of the input covariance matrix

  • alpha – Spread of the sigma points (default: 0.1)

  • beta – Prior knowledge parameter (default: 2.0)

  • kappa – Secondary scaling parameter (default: 0.0)

Returns:

a UnscentedTransformResult structure
