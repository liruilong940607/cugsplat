tinyrend::ghq

template<int N, int M, int order = 3, typename Func>
inline auto tinyrend::ghq::estimate_jacobian_and_hessian(Func const &f, glm::vec<N, float> const &mu, glm::vec<N, float> const &std_dev) -> std::pair<glm::mat<M, N, float>, std::array<glm::mat<N, N, float>, M>>

Estimates the Jacobian and Hessian of a function using Gauss-Hermite quadrature.

This function uses Gauss-Hermite quadrature points to estimate the first and second derivatives of a function at a given point. It is particularly useful for functions where analytical derivatives are difficult to compute.

Note

The function uses 27 points for order 3 and 125 points for order 5. Higher order provides better accuracy but requires more function evaluations.

Template Parameters:

  • N – Input dimension of the function

  • M – Output dimension of the function

  • Func – Type of the function to differentiate

  • order – Order of the Gauss-Hermite quadrature (default: 3)

Parameters:

  • f – The function to differentiate

  • mu – The point at which to estimate derivatives

  • std_dev – The standard deviation for each input dimension

Returns:

A pair containing:

  • The Jacobian matrix (M x N)

  • An array of Hessian matrices (M elements, each N x N)
